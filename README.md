## MyBot
AI assistant for my daily personal uses. 

#### Feature
- CLI interaction
- OpenAI GPT 
- Streaming ouput

#### Tech Stack 
- Python
- OpenAI
- HuggingFace (future development)
- LangChain (future developent)

#### Setup
Create Alias (Linux) - shortcut to run app.py file from CLI.
```
nano ~/.bashrc 
(or) 
vi ~/.bashrc
```
Inside a bashrc file create an alias to the project.
```
alias mybot='/home/{user}/mybot/.venv/bin/python /home/{user}/mybot/>

```
Execute in terminal.
```
mybot
```
